# Gold Stock Metrics - Computational Economics 

This program was developed during my undergraduate course **Computational Economics**.  
It calculates and exports several useful metrics related to **gold stock data** over time.

## Description

The program processes gold stock data and computes various financial metrics that can be useful for analysis in computational economics and financial studies.

After performing the calculations, the program exports the results for further analysis.

## How to Run

1. Download the `.exe` file and the folder "Program of Gold". (https://drive.google.com/file/d/1_R6Kg8iwqsjIt9RmKXDCuLjD68MA6CMe/view?usp=sharing)
2. Place the executable **Program of Gold*.
3. Run the `.exe` file.
4. The program will process the data and export the calculated metrics.

## Output

The program generates files containing the calculated metrics which can be used for further analysis or visualization.
